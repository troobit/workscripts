# VS Code & AI Efficiency Guide (macOS)

## `Cmd + P`
### Quick Open
Search and jump to any file by name instantly.

## `Cmd + Shift + P`
### Command Palette
Access all settings and commands (e.g., "Open Settings").

## `Cmd + B`
### Toggle Sidebar
Hide or show the file explorer tree.

## `Cmd + 0`
### Focus Sidebar
Jump focus to the file tree without a mouse.

## `Cmd + 1 / 2 / 3`
### Focus Editor Group
Switch focus between split editor panes.

## `Cmd + I`
### Copilot Inline Chat
Generate or refactor code directly at the cursor.

## `Cmd + Shift + Esc`
### Claude Code Tab
Open a dedicated Claude conversation using `~/.claudecreds`.

## `Cmd + Esc`
### Toggle AI Focus
Switch focus between the code editor and Claude panel.

### **High-Speed Habits**
* **Hide the Tree:** Use `Cmd + B` to keep your screen clean.
* **Stop Clicking:** Use `Cmd + P` for files and `Cmd + 0` for folders.
* **Contextual AI:** Use `Alt + K` to @-mention files in Claude.
* **Credential Source:** Ensure `~/.claudecreds` contains your model string for the extension to connect.
